# homebrew-effortless

Homebrew tap for [Effortless](https://github.com/iulspop/effortless-attention) — a macOS attention companion.

## Install

```
brew tap iulspop/effortless
brew install --cask effortless
```

## What is Effortless?

A macOS menubar app that helps you declare intentions and timebox focus sessions. Features a full-screen altar overlay, floating chalice widget, and session logging.
